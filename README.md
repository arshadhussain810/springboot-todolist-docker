##To-Do List Application
A full-stack To-Do List web application built using Spring Boot and Spring MVC, allowing users to efficiently manage daily tasks.

##Tech Stack
Spring MVC | Spring Boot | MySQL | Docker | Thymeleaf | Hibernate

##Features
1. Create, update, and delete tasks
2. Search and filter functionalits
3. Active & completed task counters
4. One-click Delete All Completed Tasks
5. Dockerized for easy deployment

##Run with docker
1. Run: docker compose up --build 
2. Available at:  http://localhost:8000/home
